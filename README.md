## 🚀 Overview

This project is a full-stack e-commerce application built with a strong focus on practical development concepts and real-world functionality. It leverages Firebase for authentication and backend services, with Firestore powering dynamic product data and seamless real-time updates. The goal was to create a smooth, responsive experience while implementing core features you’d expect in a production-ready app.

---

## 🔐 Authentication & Backend

- Firebase Authentication for secure user login and signup  
- Persistent sessions for a seamless user experience  
- Firestore integration for scalable and real-time data handling  

---

## 🛍️ Product & Data Handling

- Dynamic product fetching directly from Firestore  
- Efficient data flow for rendering products in real time  
- Structured database design for easy scalability and updates  

---

## 🛠️ Admin Features

- Dedicated admin functionality for managing products and data  
- Clean and controlled interface for updating store content  
- Separation between user and admin-level access  

---

## 🛒 Cart System

- Fully functional cart with automatic total cost calculation  
- Persistent cart storage across sessions  
- Real-time updates when adding or removing items  

---

## 🎨 UI & Experience

- Theme switching for a customizable user interface  
- Responsive design for smooth usage across devices  
- Clean layout focused on usability and performance  

---

## ✨ Extra Highlights

- Modular and maintainable code structure  
- Focus on real-world application logic over static design  
- Built as a hands-on project to strengthen full-stack skills
